Singel Page Movie Application(React)

This application fetches data from TMDB API and uses it to populate the related reach pages(trending, searching, popular) with help of React.

Actively used technologies are,
-React
-Bootstrap
-React Bootstrap
-CSS
-Alertify

In the future,
-Implementing "react-router-dom" and change the whole structure of the App.js with help of router technology.
-Adding a category section and fetching "genre_ids:" parameter from TMDB API to implement category management.
-Generating a specific Navi.js file as a component and replacing the present on with Navi.js. Also detailing the Navbar with new features.

How to install the project,
-npm start
